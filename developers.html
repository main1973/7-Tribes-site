Quick Start

Set Up Your Environment

1. Connect to Alkebuleum

Use standard Ethereum RPC endpoints. Alkebuleum is EVM-compatible.

RPC: https://rpc.alkebuleum.com

2. Get the 7TRB Contract ABI

Access the token contract on the block explorer or GitHub.

Contract: 0xdf7ce67dB19142672c4193d969cdD9975A5A6038

3. Integrate with Your App

Use Web3.js, Ethers.js, or your preferred library to interact with 7TRB.

const contract = new ethers.Contract(address, abi, provider);

4. Test on Testnet

Alkebuleum testnet available for development and testing.
